# Finance Tracker

A web-based personal finance tracker that helps users log income and expenses, categorize transactions, and get a clear view of their spending patterns over time.

> Live app: https://finanancetracker.vercel.app

---

## Features

- Add, edit, and delete transactions
- Separate income and expenses
- Categorize transactions (e.g. Food, Rent, Shopping, Travel)
- View transaction history in a table
- Basic summary stats (total income, total expense, balance)
- Responsive UI for desktop and mobile
- Backend API for persistent storage (separate `backend` folder)
- Environment-based configuration for local vs production
